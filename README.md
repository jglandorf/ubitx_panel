###Wiring Harness PCB Panel for the µBITX  
The (µBITX)[http://www.hfsignals.com/index.php/ubitx/] is "The QRP HF General Coverage Transceiver you can build".
 * Speeds wiring to the µBITX mic, speaker, and keyer sockets, and encoder
 * Encoder on breakaway PCB section allows for separate mounting location
 * Reduces wiring errors
 * Allows use of an inexpensive Baofeng speaker mic
 * Easily mounts to an enclosure
See docs/ubitx_panel_usage.pdf for a complete description.
Used successfully for the [Arrow Communications Assn.](https://w8rp.org/) uBitx build sessions.
